# Modern Beta Release Plus for Modern Beta Forge 1.12.2

![Modern Beta Banner](https://i.imgur.com/nhQXRx4.png)

An add-on for Modern Beta Forge. Adds biome generators from Beta 1.8 to Release 1.6.
